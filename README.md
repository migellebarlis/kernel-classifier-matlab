# Kernel Classifier

Kernel Classifier is a MATLAB source code for kernel classification of blurred images.

## Usage

For training the network

```matlab
run demo.m
```

For generating the results of the network

```matlab
run result.m
```

## Guide on image name

```matlab
image#_kernel#_feature#_filter#.png
```
